# Technologies

-   React
-   TypeScript
-   Styled components
-   Axios
-   Redux Toolkit
-   TMDB API

# Features

-   Watch the YouTube trailer
-   Check movie details
-   Add to Favorites
-   Responsive design using media queries
-   Change the main movie at regular intervals
-   Check the list of movies while sliding
